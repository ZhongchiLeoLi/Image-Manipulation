------------------------------------------------------------------------
This is a simple photo editor that applies Buffered Image, 
pixel packaging and unpackaging. User can import files to edit
and export the finishing products. This editor also features
fully functional undo and redo buttons using ArrayLists.
The types of manipulations included are:
  - Transformations:
    - Flip Horizontally/Vertically
    - Rotate 90° left/right
    - Rotate 180°
  - Colour Adjustments:
    - Red/Blue/Green -ify
    - Grey-scale
    - Negative
    - Brighten/Darken
    - Sepia Photo
------------------------------------------------------------------------

HOW TO START THIS PROJECT: Run project on Greenfoot
